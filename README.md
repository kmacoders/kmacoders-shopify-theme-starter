### Install

1. Pull từ git về :

```
npm install

```

2. Install themekit shopify

[Install Shopify Themekit o day ](https://shopify.github.io/themekit/)

### Code

Mở 2 tab terminal trên VSCode lên chạy đồng thời:

Tab 1 : 
```
npm run dev

``` 

Tab 2 : 
```
cd dist

``` 
```
theme watch

```
